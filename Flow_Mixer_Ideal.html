Flow_Mixer_Ideal

Path: CARNOT/Hydraulics


Purpose:
Model of a ideal mixing valve the flow diversion is directly set by the control input. The block feeds back the flow diversion to the Flow_Diverter.

Description:
The block mixes two massflows in an ideal way: The Ctrl signal defines the output fdiv for the Flow_Diverter block. An additional pressure drop is calculated by the Flow_Mixer_Ideal to achieve the desired fraction of the mass flow in THB1 and THB2. This additional pressure drop is added to the branch and given as pressure drop coefficients to the output THB. In the case of a closed or open mixing valve, the closed branch is not considered in the calculation. The block uses the Mix_T_Mdot block to calculate the mixture of the fluids.
See chapter 2: Basic Concepts for more details of modelling parallel flow branches in CARNOT.

Input:

THB1 : Thermo-Hydraulic Bus of branch 1
THB2 : Thermo-Hydraulic Bus of branch 2
clq : Bus with massflow and pressure drop coefficients of the THB before it was separated in THB1 and THB2 by the Flow_Diverter
Ctrl : Setpoint for the position of the mixing valve (0 = branch of THB1 is closed; 1 = branch of THB1 is opened)

Output:

THB : Thermo-Hydraulic Bus of the mixed fluid flow
fdiv : massflow diversion factor fdiv used for mixing the entering two mass flows in one mass flow, connect to the Flow_Diverter

For a correct flow diversion according to the pressure drop in the branches THB1 and THB2 connect :

  • clq input to the output of the corresponding Flow_Diverter
  • THB1 of the Flow_Diverter to the THB1 input
  • THB2 of the Flow_Diverter to the THB2 input

Mask Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

Saturation_Property

Path: CARNOT/Basic/Material_Properties


Purpose:
The block calculates the saturation properties at the specified conditions.

Description:
The block calculates the saturation properties of the (') state and the (") state, i.e. the property of boiling fluid or saturated steam according to the equation

     prop(x) = x*prop" + (1-x)*prop'

with
prop - specified property
x - vapurcontent
prop' - boiling fluid state
prop" - saturated steam state

The block calls up the function satuprop in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing
saturationproperty (temperature, pressure, fluid_type, fluid_mix)
in the Matlab Command Window. Instead of the italic variables the actual value of the respective property has to be entered.
For more information about the function type:
help saturationproperty
The property is specified by entering a property_ID (see Parameters and Dialog Box. In the same way the phase has to be entered in the mask of the block to determine which saturationproperty, water or steam, should be calculated.

Input:

T : temperature (scalar or vector) in °C
p : pressure in Pa
Fluid_Type : see definition of CARNOT fluid types
Fluid_Mix : see definition of CARNOT fluid mixtures

Output:

density : density in kg/m3

Parameters and Dialog Box:

You can choose the inputs to be the :

  • state variables (temperature, pressure, fluid type and mix have separate input ports)
  • the Thermo Hydraulic Bus (THB)
  • or, for compatibility with older versions, the Thermo Hydraulic Vector (THV)

Use the GlobalMessageLevel block to handle the debugging messages.

Examples:
Open the example explorer from the Matlab command window
ExampleBrowser
or load the examples via the CARNOT library.

  • exampleSaturationProperty

NOTE:
The saturation_property block is only available for water.

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

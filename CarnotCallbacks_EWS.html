Description

Callback of the Ground_Source_Heat_Exchanger model in Carnot
For geothermal probe fields it is assumed that all probes are connected
in parallel.
The temperatures the ground ('earth')are split in a 'near probe' area
with radial and axial nodes in a cylinder geometry and a 'far probe'
area where ground temperature is calculated by the g-function.
 Tearth(i,1)         radial node 1 (filling) in axial layer i
 Tearth(i,DimRad)    radial node DimRad (outer radial node) in axial layer i
 Tearth(i,DimRad+1)  radial node temperature calculated by g-function
State space model
 dTearth(i,j) = Ar * Tearth(i,j) + Br * [Tearth(1,0); Tearth(i,DimRad+1)]
 Tearth(i,j) = Cr * Tearth(i,j) + Dr * [Tearth(1,0); Tearth(i,DimRad+1)]
The brine flow is cacluated for each axial layer:
Flow downwards (state space model)
 dTd(i)/dt = Ad * Td(i) + Bd * [Tin; Tearth(i,1)]
 Td(i)     = Cd * Td(i) + Dd * [Tin; Tearth(i,1)]
Flow upwards (state space model)
 dTu(i)/dt = Au * Tu(i) + Bu * [Td(DimAxi); Tearth(i,1)]
 Tu(i)     = Cu * Tu(i) + Du * [Td(DimAxi); Tearth(i,1)]
details of the mathematical model see Ochs_Carnot_EWS_2012.pdf
and Huber 1999

Pressure_Drop

Path: CARNOT/Basic/Pressure_Drop



Purpose:
The block calculates the pressure drop in a hydraulic component.

Description:
The block calculates the pressure drop in according to the inputs by the following equation:
       dp = con + lin*mdot + qua*mdot2
The block is used in the pipe model of CARNOT.

Input:

THBin : incomming Thermo Hydraulic Bus
con : constant pressure drop coefficient in Pa
lin : linear pressure drop coefficient in Pa/(kg/s)
qua : quadratic pressure drop coefficient in Pa/(kg/s)2

Output:

THB : leaving Thermo Hydraulic Bus

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

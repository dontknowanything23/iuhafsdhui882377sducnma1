Min_Run_Time

Path: CARNOT/Control

Purpose:
Keep a positive control signal for a minimum operation time.

Description:
Once the signal changes from zero to a positive value, the output is kept to the maximum of the input signal and the minimum output value during the minimum operation time. If the input is set to zero during the minimum operation time the output set to a Minimum output value. After the minimum operation time the output signal is equal to the input signal.

Input:

In1 : control signal (no unit)

Output:

Out1 : control signal (no unit)

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

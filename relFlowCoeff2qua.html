relflowcoeff2qua

Path: CARNOT/Basic/Pressure_Drop


Purpose:
The blocks convert the relative flow factor Psi to the Carnot pressure drop coefficient q.

Description:
Determine the Carnot pressure drop coefficient q (or qua) from the flow coefficient Kv.
The flow coefficient Kvs for an open valve is the volume flow rate which creates a pressure drop of 1e5 Pa (1 bar). So for any position of the valve the flow coefficient Kv is :
    Kv = Vdot * sqrt(1e5Pa/dp * rho/1000kg/m3)    (equation 1)

    Vdot = mdot/rho     (equation 2)

with
Vdot : volumetric flow rate in m3/h
rho : density in kg/m3
Kv : flow coefficent in m3/h

The Carnot equation for pressure drop is :
    dp = c + l*mdot + q*mdot2     (equation 3)

For valves the flow coefficient for a fully open valve Kvs in m3/h is given. With the relative flow factor Psi the flow coefficent Kv is:
    Kv = Psi * Kvs

Resolve (equation 1) for pressure drop dp
    dp = (100 Pa*m³/kg)/rho * (mdot/Kv)2

change unit of Kv: kg/h = kg/(3600s)
    dp = (100 Pa*m³/kg) * (3600 s/h)2 / rho * (mdot/Kv)2
    q = (100 Pa*m³/kg) * (3600 s/h)2 / (rho*Psi2*Kvs2)

For more information see the density block and the velocity block.

Input:

THB : Thermo-Hydraulic Bus
psi : dimensionless relative flow factor

Output:

qua : Carnot pressure drop coefficient q in Pa/(kg/s)2

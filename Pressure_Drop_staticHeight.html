Pressure_Drop_staticHeight

Path: CARNOT/Basic/Pressure_Drop



Purpose:
Calculate the static pressure drop or increas due to a difference in static height.

Description:
Calculate pressure drop or increase due to statig height:

     dp = height / N_nodes * grav * density(Tnodes)

The length of "Tnodes" vector defines the number of temperature nodes. The height is between inlet and outlet. It is divided by the number of nodes Nnodes in the temperature vector "Tnodes". Height is positive for outlet above inlet when pressure decreases.
Calculation is done for Flow_ID > 10000, static height only for Flow_ID > 20000, see Basic Concepts for details.

Input:

Tnodes : node temperature or temperatures of the pipe in °C
THBin : inlet Thermo Hydraulic Bus
height : difference of height between inlet and outlet in m
(positive for outlet above inlet)

Output:

THB : outlet Thermo Hydraulic Bus

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

Display_THB

Path: CARNOT/Outputs/Selectors



Purpose:
The block shows the elements in the Thermo-Hydraulic Bus.

Description:
The block shows the temperature, mass flow rate, pressure, fluid type, fluid mixture or diameter of up to six Thermo-Hydraulic Buses. In the parameter block the physical value can be chosen. The number of inputs can be changed by changing the blocks in the subsystem. Be sure to short all unconnected outputs to the Terminator block to avoid a warning in the MATLAB command window.

Output:

to_scope : selected value in the Thermo-Hydraulic Bus

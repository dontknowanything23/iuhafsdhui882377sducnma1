Limit_Power_Change

Path: CARNOT/Control


Purpose:
Limit a control signal to the boundary conditions of power change during startup.

Description:
The block limits an incomming signal during startup:

  • input signal of zero : output signal is set to zero
  • input signal above zero : output signal start with "control signal at start", than changes to the minimum of the control signal input and the "max. control signal before reaching min. temperature" until Ts reaches the minimum temperature.
The change of the output signal is limited to maximum of power increase / decrease.

Input:

ctrl : control signal input
Ts : measured temperature (typically the supply temperature) in °C

Output:

Ctrl : control signal output

Description

CarnotCallbacks_Data_from_File is a callback script to load files and
write the filename in a From_Workspace Block.
The variable pathname can be used to hand over the path to files which
are not stored on the Matlab-path.
The data needs to have the timestamp in the 1st column. The function
assumes that the timerange of the data is one year, starting first of
january.
For simulation of more than one year the data of the file is repeated to
cover the simulation time if necessary. This works for positive stop
times above 8760*3600 s and also for negative start times. A start time
of -30*24*3600 for example is possible.
The data is extended in the following way:
1. If the data has no timestamp 0 s at the beginning, the first line of
   the data is copied added at the start of the data. A timestamp of 0 s
   is added for this line.
2. If the data has no timestamp of 365*24*3600 s, the last line of
   the data is copied added at the end of the data. A timestamp of
   365*24*3600 s is added for this line.
3. For a start time before 0 s, the complete data is duplicated with a
   the original timestamp - 365*24*3600 s. Any data lines before start
   time are not duplicated in the data.
4. For a stop time later than 365*24*3600 s, the complete data is
   duplicated with the original timestamp + 365*24*3600 s. Any data
   lines before after stop time are not duplicated in the data.

extendAnnualData(Data, start_time, stop_time) - extend data to the full simulation timerange

Function Call
out = extendAnnualData(Data, start_time, stop_time)
Inputs
Data         - matrix with input Data without extension
start_time   - start time of the simulation in s
stop_time    - stop time of the simulation in s
Outputs
out         - matrix with extended Data if necessary
Description
extend data to the data of the simulation time if necessary

extend the data to one year if needed

add timesteps at the beginning or end if needed

start time is smaller than first timestamp in the Data

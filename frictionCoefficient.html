clq2friction_coeff, friction_coeff2clq

Path: CARNOT/Basic/Pressure_Drop


Purpose:
The blocks convert the friction coefficent to the Carnot clq coefficients and vice versa.

Description:
Determine the pipe friction coefficent lambda from Carnot clq with the block clq2friction_coeff or determine the clq coefficients from pipe friction coefficent lambda with the block friction_coeff2clq.
The pressure drop dp for a given friction coefficient lambda is:

     dp = lambda * length * rho / (dia * 2) * v

with:
rho : density in kg/m3
dia : diameter im m

For round pipes:
     dp = 8 * lambda * l * mdot2 /(rho * pi2 * dia5)

Carnot pressure drop calculation:
     dp = c + l*mdot + q*mdot2

As the equation with lambda is quadratic in the velocity (~mass flow), the constant and linear coefficients of the Carnot equation are zero.
If lambda is given, q can be determinded with:
     q = 8 * lambda * l / (rho * pi2 * dia5)

If the Carnot quadratic pressure drop coefficient q is given, lambda can be calculated:
     lambda = q * rho * pi2 * dia5 / (8 * l)

Input:

THB : Thermo-Hydraulic Bus
clq2friction_coeff: q : quadratic pressure drop coefficient in Pa/(kg/s)2
friction_coeff2clq: pipe_friction_coeff : for : pipe friction coefficient lambda in -

Output:

clq2friction_coeff: pipe_friction_coeff : pipe friction coefficient lambda in -
friction_coeff2clq: q : quadratic pressure drop coefficient in Pa/(kg/s)2

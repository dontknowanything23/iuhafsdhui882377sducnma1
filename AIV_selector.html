AIV_Selector

Path: CARNOT/Outputs/Selectors/AIV_selector

Purpose:
Selector for Components from the Air Infiltration Vector AIV.

Description:
This block delivers a user specified vector with elements of the [AIV] vector. The sequence of elements in the output vector is identical to the sequence of selected check boxes counted from to the top to the bottom.

Input:

[AIV] : vector from Room-NODE block

Output:

[AIV]s : output vector with user selected elements from [AIV]

Parameters and Dialog Box:

NOTE:
The AIV is created by older version of the room node in CARNOT. It has been replaced by the AIB, the Air Infiltration Bus. For the description see AIB

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

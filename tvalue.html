function call

tvalue(time, refyear)

Input

time        is the time in seconds
refyear     is the reference year for the output YYYY
All input parameters are optional. If not specified, a dialog box will
open.

Output

date (integer value) in the format YYYYMMDDHHMM

Description

The function creates a matrix of chars with the Carnot timecomment in
the weather data files.
see also wformat

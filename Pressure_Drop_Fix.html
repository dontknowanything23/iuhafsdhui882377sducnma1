Pressure_Drop_Fix

Path: CARNOT/Basic/Pressure_Drop

Purpose:
Calculate the pressure drop presuming a quadratic dependency from the mass flow rate.

Description:
The block calculates the pressure drop depending on the mass flow rate. A quadratic function is used.

     dp = CONSTANT + LINEAR * mdot + QUADRATIC*mdot2

Input:

THB : incomming Thermo Hydraulic Bus

Output:

THB_ : leaving Thermo Hydraulic Bus

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

Function call

[inlet, outlet, up, down, in, idx_in_out, flowdir, hh] = ...
   CarnotCallbacks_PortsStorage(nodes, h_in, h_out, hgt);

Inputs

bhandle  : block handle, variable with the result of gcbh
nodes    : number of storage nodes
h_in     : relative height if inlet in 0..1 (0 = bottom, 1 = top)
h_out    : relative height if oulet in 0..1 (0 = bottom, 1 = top)
hgt      : height of store in m

Outputs

inlet    : node number of the port inlet
outlet   : node number of the port outlet
up       : vector of length nodes, set to 1 if nodes have a massflow
           upwards due to massflow entering by the port
down     : vector of length nodes, set to 1 if nodes have a massflow
           downwards due to massflow entering by the port
in       : vector of length nodes, set to 1 at the position of the inlet
idx_in_out : vector of length nodes, set to 1 for all nodes between
           inlet and outlet of the port
flowdir  : set to 1 for inlet below outlet,
           set to -1 for inlet above outlet
hh       : character string with the height between inlet and outlet in m

Description

Sets flow direction, inlet and outlet postion of the storage port
models. In the store model, the lowest node is node 1, the top node has
the highest number. For example if the storage model has 10 nodes,
the inlet and outlet are defined as following:
Node     Relative height h_in / h_out
10       > 0.9
 9       > 0.8 .. >= 0.9
 8       > 0.7 .. >= 0.8
 7       > 0.6 .. >= 0.7
 6       > 0.5 .. >= 0.6
 5       > 0.4 .. >= 0.5
 4       > 0.3 .. >= 0.4
 3       > 0.2 .. >= 0.3
 2       > 0.1 .. >= 0.2
 1                <= 0.1
Used in the following blocks: Pipe_Connection, Pipe_Stratified,
HX_EN12977, HX_EN12977_stratified, HX_data_fit, HX_data_fit_stratified,
HX_finned_tube, HX_finned_tube_stratified, HX_tube, HX_tube_stratified

main function

check inputs, if they are empty at start of the model

Vapour_Pressure

Path: CARNOT/Basic/Material_Properties


Purpose:
Calculates the vapour-pressure of the specified material.

Description:
This block calculates the vapour-pressure [Pa] of saturated steam according to the inputs. In the case of pure water (fluid ID 1) the function vapourpressure calculates the values of saturated steam. In the case of moist air (fluid ID 2) the function vapourpressure calculates the partial pressure of the water in the moist air. The vapourpressure is defined between the triple point at 612 Pa and the critical point at 22.12 MPa.
The block calls up the function vapourpressure in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing
vapourpressure(temperature, pressure, fluid_type, fluid_mix)
in the Matlab Command Window. Instead of the italic variables the actual value of the respective property has to be entered.
For more information about the function type:
help vapourpressure

Input:

T : temperature (scalar or vector) in °C
p : pressure in Pa
Fluid_Type : see definition of CARNOT fluid types
Fluid_Mix : see definition of CARNOT fluid mixtures

Output:

density : density in kg/m3

Parameters and Dialog Box:

You can choose the inputs to be the :

  • state variables (temperature, pressure, fluid type and mix have separate input ports)
  • the Thermo Hydraulic Bus (THB)
  • or, for compatibility with older versions, the Thermo Hydraulic Vector (THV)

Use the GlobalMessageLevel block to handle the debugging messages.

UnitConv_Temperature

Path: CARNOT/Toolbox

Purpose:
The block converts a temperature from a unit to another.

Description:
Conversion of temperature units. The availlable units are:

  • degree Celsius (°C)
  • Kelvin (K)
  • degree Fahrenheit (F)

Input:

Tinit : input temperature in a selected unit

Output:

Tfinal : output temperature in a selected unit

Parameters and Dialog Box:

Examples:
Open the example explorer from the Matlab command window
ExampleBrowser
or load the examples via the CARNOT library.

NOTE:
Do not convert temperature differences from °C to K.

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

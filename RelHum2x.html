RelHum2x

Path: CARNOT/Basic/Material_Properties


Purpose:
The block converts the relative humidity to absolute humidity.

Description:
The function converts the relative humidity to the absolute humidity in mass of water per mass of dry air, i.e the load of water in moist air according to the inputs. The material properties functions can also be called directly from the command window by typing
rel_hum2x(temperature, pressure, rel_hum)
in the Matlab Command Window. Instead of the italic variables the actual value of the respective property has to be entered.
For more information about the function type:
help rel_hum2x

Input:

T : temperature (scalar or vector) in °C
p : pressure in Pa
rel_hum : relative humidity in percent (0..100)

Output:

x : absolute humidity in kgwater/kgair

This block is only to be used in the case of air (fluid type 2, see FluidEnum).

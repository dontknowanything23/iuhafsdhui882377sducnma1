Count_Working_Time

Path: CARNOT/Toolbox

Purpose:
The block counts the operatin time and cycles of the input.

Description:
The block gives the actual operation time (= time when input is above zero), the total working time (= integral working time since start of the simulation) and the number of on cycles (= number of events when input changes from 0 to a value above 0).

Input:

input : signal to be evaluated
- - - : - - -

Output:

flank : flank is set to 1 when the input rises from 0 to a value above 0, flank is 0 otherwise
cycles : number of on cycles since start of the simulation
actual worktime : operation time of the current "on-event" in s
total worktime : operation time since since start of the simulation in s

AIV_WetBulbTemperature

Path: CARNOT/Basic/Material_Properties


Purpose:
The block calculates the wet bulb temperature.

Description:
The function uses a look-up table created by the function rel_hum2x to calculate the wet bulb temperature in °C of moist air according to the input.
This block is only useful in the case of moist air (fluid type = 2).

Input:

AIB : Air Infiltration Bus see also CreateAIB

Output:

WBT : Wet Bulb Temperature in °C
rel_hum : relative humidity in %

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

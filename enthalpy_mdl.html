Enthalpy

Path: CARNOT/Basic/Material_Properties




Purpose:
This block calculates the specific enthalpy [J/kg] of the specified material.

Description:
This block calculates the specific enthalpy in [J/kg] according to the inputs. The block calls up the function enthalpy in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing
enthalpy(temperature, pressure, fluid_type, fluid_mix)
in the Matlab Command Window. Instead of the italic variables the actual value of the respective property has to be entered.
For more information about the function type:
help enthalpy
For the calculation of steam, the properties are calculated according to E.Schmidt: Properties of water and steam [Schmidt, 1982]. The steam properties can be calculated for a temperature range up to 800°C and a pressure range up to 49.055 MPa. The calculations of the liquid phase and the saturation states are performed with the polynomials from B.Glück: Stoffwerte und Zustandsgrößen [Glück, 1991]. The valid temperature range is up to 200°C and the pressure range is up to 2 MPa. The enthalpy of the liquid phase considers only the dependency of the temperature, the pressure dependency is neglected.
The enthalpy is only available for the fluids water (fluid type 1) and air (fluid type 2).

The valid range is for the respective value is:

fluid_ID fluid phase temperature pressure
1 water vaporous T < 800 °C p < 49 MPa
1 water liquid 0 °C < T < 200 °C 100 kPa < p < 10 MPa
2 air dry -150 °C < T < 1000 °C 0.1 MPa < p < 10 MPa
2 air moist 0 °C < T < 800 °C 0.1 MPa < p < 10 MPa

Input:

T : temperature in °C
p : pressure in Pa
Fluid_Type : see definition of CARNOT fluid types
Fluid_Mix : see definition of CARNOT fluid mixtures

Output:

Enthalpy : specific enthalpy of the fluid in J/kg

Parameters and Dialog Box:

You can choose the inputs to be the :

  • state variables (temperature, pressure, fluid type and mix have separate input ports)
  • the Thermo-Hydraulic Bus THB
  • or, for compatibility with older versions, the Thermo Hydraulic Vector (THV)

Use the GlobalMessageLevel block to handle the debugging messages.

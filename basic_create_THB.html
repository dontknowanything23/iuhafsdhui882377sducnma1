basic_create_THB

Path: CARNOT/Basic/Hydraulics

Purpose:
Create a Thermo-Hydraulic Bus with a variable massflow and temperature.

Description:
The output of the block is a Thermo-Hydraulic-Bus. The temperature and the massflow are defined by an input port.

Input:

T : temperature in °C
mdot : mass flow in kg/s

Output:

THB : Thermo Hydraulic Bus

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

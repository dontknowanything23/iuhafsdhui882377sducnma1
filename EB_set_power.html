EB_set_power

Path: CARNOT/Source/Simple

Purpose:
Set power in the Electric Bus.

Description:
The block sets the active power of all three lines in the Electric Bus. The Block overwrites existing values of the active power in the incomming bus. Feed in a vector with three elements to adress each line individually.

Input:

EB_in : incomming Electric Bus
act_power : active power of the lines (vector with 3 elements)

Output:

EB : leaving Electric Bus with the new values for the active power

Examples:
Open the example explorer from the Matlab command window
ExampleBrowser
or load the examples via the CARNOT library.

Direct Feedthrough : Yes

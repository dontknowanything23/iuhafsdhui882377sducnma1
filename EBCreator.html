EBCeator

Path: CARNOT/Basic/Electric
















Purpose:
Create a constant Electric Bus.

Description:
The output of the block is a constant Electric Bus signal.

Input:

id : electric bus identifier
Frequency1 : frequency phase 1 in Hz
Voltage1 : voltage phase 1 in V
ActivePower1 : active power phase 1 in W
PowerFactor1 : power factor phase 1
Frequency2 : frequency phase 2 in Hz
Voltage2 : voltage phase 2 in V
ActivePower2 : active power phase 2 in W
PowerFactor2 : power factor phase 2
Frequency3 : frequency phase 3 in Hz
Voltage3 : voltage phase 3 in V
ActivePower3 : active power phase 3 in W
PowerFactor3 : power factor phase 3

Output:

EB : Electric Bus

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

converts a given solar time to an hourangle

An hourangle of 0° means 12:00 solar time

Contents

  • Function call
  • Input
  • Output
  • Copyright and Versions

Function call

hourangle = time2hourangle(time)

Input

solar time [s]

Output

hourangle in °, range [-180,180], west negative = afternoon
see also: hourangle2time, legaltime2solartime, solartime2legaltime

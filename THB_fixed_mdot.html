THB_fixed_mdot

Path: CARNOT/Source/Simple

Purpose:
Set the mass flow in the Thermo-Hydraulic Bus.

Description:
The block replaces the mass flow in the Thermo-Hydraulic Bus with the new value specified as parameter in the dialog box.

Input:

THB_in : incomming Thermo-Hydraulic Bus

Output:

THB : Thermo-Hydraulic Bus with the new value for the mass flow

Pressure_Drop_StoragePipe

Path: CARNOT/Basic/Pressure_Drop



Purpose:
Calculate the pressure drop of a pipe connection to a storage.

Description:
Calculate pressure drop of a pipe connection to a storage depending on the parameters:

  • the diameter increase at the inlet
  • the diameter reduction at the outlet
  • the static pressure change to to height difference
  • pressure drop of a pipe connection to the store

For details of the calculations see Pressure_Drop_changeD and Pressure_Drop_staticHeight.
Calculation is done for Flow_ID > 10000, static height only for Flow_ID > 20000, see Basic Concepts for details.

Input:

Tnodes : node temperature or temperatures of the store in °C
THBin : incomming Thermo Hydraulic Bus
positon : positon vector of the collector, see Fixed_Surface

Output:

THB : leaving Thermo Hydraulic Bus

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

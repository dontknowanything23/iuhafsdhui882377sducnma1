Temperatures_Limits_CHP

Path: CARNOT/Control

Purpose:
Limit a control signal to a flow and return line temperature range.

Description:
The block limits an incomming signal to a lower and upper limit of the return line and the flow line. The temperatures have to be in the valid range after the startup time.

Input:

ctrl : control signal input
Tr : return line temperature in °C
Ts : supply line temperature in °C

Output:

Ctrl : control signal output

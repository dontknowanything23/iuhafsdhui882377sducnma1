basic_create_EB

Path: CARNOT/Basic/Electric





Purpose:
Create an Electric Bus.

Description:
Active power and voltage are set by input ports. The other parameters are constant and set in the block mask. The output of the block is a constant Electric Bus signal.

Input:

voltage : voltage DC or AC all phases in V
ActivePower1 : active power DC or AC all phases in W

Output:

EB : Electric Bus

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

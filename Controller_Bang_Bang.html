Controller_Bang_Bang

Path: CARNOT/Control

Purpose:
Hysteresis controller for thermal systems.

Description:
The block is a simple hysteresis controller. The outlet is set to:

1 : if (Thot - Tcold) > deltaTon
0 : if (Thot - Tcold) < deltaToff

The output is always set to zero if Thot or Tcold > Tmax

Input:

Thot : temperature e.g. of the solar thermal collector in °C
Tcold : temperature e.g. of the thermal store in °C

Output:

control : control signal (0 or 1)

Evaporation_Enthalpy

Path: CARNOT/Basic/Material_Properties >


Purpose:
This block calculates the evaporation enthalpy [J/kg] of the specified material.

Description:
This block calculates the evaporation enthalpy in [J/kg] according to the inputs. The evaporation enthalpy r is the enthalpy that is necessary to evaporate the material. It is calculated by the difference of the saturation properties of liquid (' state) and vapourous phase (" state) at the saturation point. At the saturation point, the liquid phase is boiling fluid and the vapourous phase is saturated steam.
The block calls up the function evaporation_enthalpy in the CARNOT Carlib Library where the calculation is carried out. The material properties functions can also be called directly from the command window by typing
evaporation_enthalpy(temperature, pressure, fluid_type, fluid_mix)
in the Matlab Command Window. Instead of the italic variables the actual value of the respective property has to be entered.
For more information about the function type:
help evaporation_enthalpy
The calculation of the saturation states are performed with the polynomials from B.Glück: Stoffwerte und Zustandsgrößen [Glück, 1991]. As the saturationvalues only exist for water the evaporation_enthalpy is also only available for the fluid water (fluid type 1). The valid temperature range is up to from the temperature 10°C up to 370°C.

Input:

T : temperature in °C
p : pressure in Pa
Fluid_Type : see definition of CARNOT fluid types
Fluid_Mix : see definition of CARNOT fluid mixtures

Output:

evaporation_enthalpy : evaporation enthalpy of the fluid in J/kg

Parameters and Dialog Box:

You can choose the inputs to be the :

  • state variables (temperature, pressure, fluid type and mix have separate input ports)
  • the Thermo Hydraulic Bus (THB)
  • or, for compatibility with older versions, the Thermo Hydraulic Vector (THV)

Use the GlobalMessageLevel block to handle the debugging messages.

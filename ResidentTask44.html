Resident_Task44
Path: CARNOT/Loads/Houses

Purpose
Model of the internal gains, emissions, the window ventilation and the shading control for the House_simple model as defined by the Task 44.

Description

A full description is available at [Dott, Haller, Ruschenburg, Ochs, Bony : The Reference Framework for System Simulations of the IEA SHC Task 44 / HPP Annex 38 Part B: Buildings and Space Heat Load  2013].

Input:

  • WDB : Weather Data Bus
  • Troom : Room temperature in °C

Output:

  • RDB : Resident Data Bus with
    • Qdot_internal : internal gains in W
    • Sunshade : shading of the windows (0 = no shading, 1 = full shading)
    • OpenWindow : opening of the window (0 = closed, 1 = opened), parameters for the opening angle and ventilation are defined in the window model
    • Massflow of CO2: generated massflow by internal sources (calulculation is only done when CalculateAirComposition is true)
    • Massflow of H2O: generated massflow by internal sources  (Not considered by the House_simple model as it has no humidity model)
    • Massflow of CO2: generated massflow by internal sources  (calulculation is only done when CalculateAirComposition is true)

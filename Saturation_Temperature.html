Saturation_Temperature

Path: CARNOT/Basic/Material_Properties


Purpose:
Calculates the saturation temperature of the moist air at the specified conditions.

Description:
Calculates the saturation temperature of moist air ( fluid_ID = 2 ) or pure water ( fluid_ID = 1 ) in [°C] according to the inputs.
The block calls the function "saturationtemperature" in the CARNOT Carlib Library where the calculation is carried out.

The material properties functions can also be called directly from the command window by typing
saturationtemperature(temperature, pressure, fluidtype, fluidmix)
Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume. For more information about the function type
help aturationtemperature
to the workspace as for any other MATLAB function.

In the case of pure water (fluid_ID = 1) the function saturationtemperature calculates the temperature of saturated steam at the specified pressure.
In the case of moist air (fluid_ID = 2) the function saturationtemperature calculates the dew point temperature of the water in the moist air.
See definition of the fluid types above.
The valid range of the saturationtemperature is below the critical pressure (p < pcrit= 221.2 bar).

Input:

T : temperature of the fluid in °C
p : pressure in Pa
Fluid_Type : fluid type, see FluidEnum for the definition
Fluid_Mix : fluid mixture, see FluidEnum for the definition

Output:

Tcond : temperature conductivity in m2/s

NOTE:
The saturation temperature is only available for water and moist air.

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

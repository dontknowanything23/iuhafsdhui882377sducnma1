Wait

Path: CARNOT/Toolbox

Purpose:
Slow down the simulation to real time.

Description:
This block pauses the simulation until the system time reaches the specified date. After the first call, the block pauses the simulation for the specified number of seconds. When the block appears more than once in a simulink model and each waiting time is e.g. 2 seconds, the function will only wait once 2 seconds per simulation step. If the blocks have different waiting times, the simulation will pause in each simulation step for the longest specified waiting time. For Real Time applications with an interface to the environment which requires the system time instead of the simulation time to evaluate external data.

NOTE:
Simulink for Matlab R2018b offers the "Pacing Options" (Menue "Simulation"). With this option you may slow down Simulink to real time (more or less).
The use of this option should be considered before using the "Wait" block.

Characteristics:

Direct Feedthrough : No
Sample Time : Specified by the parameter "waiting time"
Vectorized : No

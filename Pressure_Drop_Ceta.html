Pressure_Drop_Ceta

Path: CARNOT/Basic/Pressure_Drop


Purpose:
The block calculates the pressure drop with a pressure loss coefficient.

Description:
The block calculates the pressure drop in a pipe according to the parameters:

  • temperature in °C
  • velocity in m/s
  • Fluid type (see fluid types)
  • Fluid mixture (see fluid types)
  • inner pipe diameter in m
  • pressure loss coefficient Ceta
  • density of the fluid rho in kg/m3

The equation for the pressure drop is:

       dp = Ceta * rho/2 * velocity²

The block uses the pressuredrop2clq block.
Calculation is done for Flow_ID > 10000, see Basic Concepts for details.
See also the density block and the velocity block.

Input:

THBin : incomming Thermo Hydraulic Bus
dp_coeff : pressure loss coefficient Ceta

Output:

THB : leaving Thermo Hydraulic Bus

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

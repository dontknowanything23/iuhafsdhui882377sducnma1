GlobalMessageLevel

Path: CARNOT/Toolbox


Purpose:
Handle the error messages of the fluid property blocks.

Description:
The block handles the warning messages in the background via the CarnotCallbacks_GlobalMessageLevel and the MessageLevelEnum. Messages and warnings can be saved in a text-file. The message settings can be :

  • local = each block determines its error handling (the parameters are only accessible by the fluid property blocks)
  • global = the GlobalMessageLevel block determines the error handling for all blocks in a model (parameters are set by the GlobalMessageLevelBlock)

You can parameterize the warnings and error messages of the fluid property blocks:

  • Density
  • Enthalpy
  • Heat_Capacity
  • Thermal_Conductivity

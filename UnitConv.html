UnitConv

Path: CARNOT/Toolbox

Purpose:
Converts a value from a specified unit to another unit.

Description:
The symbols in a specified unit have to be separated by * or /. So kWh has to be written as kW*h. The common signs are allowed, e.g. kW, mm, µJ. Beside the units this functions knows some constants, e.g. when using the "unit" c, you get the lightspeed.

The Block UnitConv_Temperature is a special case of unitconv, witch convert only temperatures between the three units: °C, °F and K. This calculation is also available as a Matlab function: unitconv_temperature.

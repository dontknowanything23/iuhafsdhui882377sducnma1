Function Call

y = cs_energy_heatpump([timeclass], [file], [filedetail])

Inputs

timeclass - optional input parameter for the evaluation period
 1 : year (default)
 2 : month
 3 : week
 4 : day
 5 : hour
file - optional character string with the name of the .mat file
 created by the data logger. If not given during function call, the
 function itsself will open an input window for the choice of the file.
 Files written by the Logger_HeatPump have the format 'heatpumpoutput*.mat'
filedetail - optional character string with the name of the .mat file
 created by the enabled data logger in the Logger_HeatPump block.
 This file contain the same information as file but they are stored only
 when the heat pump is working.
 Files written by the Logger_HeatPump have the format 'heatpumpdetailoutput*.mat'
file and, if given filedetail, must contain the following information in timeseries:
 hp.Tc_in        source inlet temperature in degree Celsius
 hp.Tc_out       source outlet temperature in degree Celsius
 hp.MassC        source integrated massflow in kg
 hp.Qc           source energy in Joule
 hp.Th_in        heating inlet temperature in degree Celsius
 hp.Th_out       heating outlet temperature in degree Celsius
 hp.MassH        heating integrated massflow in kg
 hp.Qh           heating energy in Joule
 hp.Welec_in     consumed electric energy in Joule
 hp.Cycles       compressor number of 'on' cycles
 hp.OpTime       compressor working time in s
 hp.CostElecBuy  costs of the electric energy in 1/J

Outputs

y - matrix with columns:
  1   time of simulation or timeclass start
  2   time of simulation or timeclass end
  3   heating circuit energy in Joule
  4   consumed electric energy in Joule
  5   source energy in Joule
  6   SPF - seasonal (or timeclass period) performance factor
  7   number of on/off cycles
  8   operation time in seconds
  9   average source massflow in kg/s
 10   average heating circuit massflow in kg/s
 11   mean source inlet temperature in degree Celsius       *)
 12   mean source outlet temperature in degree Celsius      *)
 13   mean heating circuit inlet temperature degree Celsius *)
 14   mean heatingcircuit outlet temperature degree Celsius *)
 15   electric energy costs in 1/J
 *) The average values are only calculated when 'filedetail' is given.
   The values are evaluated at the operation time of the compressor only.

Description

Evaluation of data written by the Logger_Heatpump. The result of the
function is the energy in annual, monthly, ... values, depending
on the timeclass. For timeclass 1 the first row has the annual values.
For timeclass 2 the 12 rows contain the monthly values.
The outputs are organized in groups: time, energy, efficiency,
operation, average massflows, average temperatures, economics

Function Call

y = cs_energy_energymeter([timeclass], [file], [filedetail])

Inputs

timeclass - optional input parameter for the evaluation period
 1 : year (default)
 2 : month
 3 : week
 4 : day
 5 : hour
file - optional character string with the name of the .mat file
 created by the data logger. If not given during function call, the
 function itsself will open an input window for the choice of the file.
 Files written by the Logger_Energy_Meter have the format
 'energymeteroutput*.mat'
filedetail - optional character string with the name of the .mat file
 created by the enabled data logger in the Logger_Energy_Meter block.
 This file contain the same information as file but they are stored only
 when the there is a massflow.
 Files written by the Logger_Energy_Meter have the format
 'energymeterdetailoutput*.mat'
file and, if given filedetail, must contain the following information
in timeseries:
 meter.T_in : inlet temperature in degree centigrade
 meter.T_out : outlet temperature in degree centigrade
 meter.Mass : integrated massflow in kg
 meter.Cycles : number of 'on' cycles
 meter.OpTime : working time in s
 meter.Q : energy in Joule

Outputs

y- Matrix where for timeclass 1 the first row has the annual values.
 For timeclass 2 the 12 rows contain the monthly values.
Columns:
 1   time of simulation or timeclass start
 2   time of simulation or timeclass end
 3   energy from or to the system in Joule (positive values for T_out > T_in)
 4   number of on/off cycles
 5   operation time in seconds
 6   average massflow in kg/s
 7   mean inlet temperature during operation in degree Celsius *)
 8   mean outlet temperature during operation in degree Celsius *)
 *) The average values are only calculated when 'filedetail' is given.
   The values are evaluated at the operation time only (time with massflow).

Description

Evaluation of data written by Logger_Heatpump.The result of the
function is the energy in annual, monthly, ... values, depending
on the timeclass. For timeclass 1 the first row has the annual values.
For timeclass 2 the 12 rows contain the monthly values.
The outputs are organized in groups: time, energy, efficiency,
operation, average massflows, average temperatures, economics

Function Call

y = cs_energy_house([timeclass], [file], [filedetail])

Inputs

timeclass - optional input parameter for the evaluation period
 1 : year (default)
 2 : month
 3 : week
 4 : day
 5 : hour
file - optional character string with the name of the .mat file
 created by the data logger. If not given during function call, the
 function itsself will open an input window for the choice of the file.
 Files written by the Logger_House have the format 'houseoutput*.mat'
filedetail - optional character string with the name of the .mat file
 created by the enabled data logger in the Logger_House block.
 This file contain the same information as file but they are stored only
 when there is a massflow through the heating system.
 Files written by the Logger_House have the format 'housedetailoutput*.mat'
file and, if given filedetail, must contain the following information in timeseries:
 house.Temperature_Ambient : ambient temperature in degree Celsius
 house.Troom               : room temperature in degree Celsius
 house.Troom_setpoint      : setpoint for room temperature in degree Celsius
 house.Qh                  : heating energy from the system to the house in Joule
 house.Qc                  : cooling energy from the system to the house in Joule
 house.Qloss               : transmission losses in Joule (negative value for losses)
 house.Qvent               : ventilation losses in Joule (negative value for losses)
 house.Qinternal           : internal gains in Joule (positive value for gains)
 house.Qsolar              : solar gain in Joule (positive value for gains)
 house.T_in                : inlet temperature of the heating sytem in degree Celsius
 house.T_out               : outlet temperature of the heating sytem in degree Celsius
 house.Mass                : integrated massflow through the heating system in kg
 house.Cycles              : number of on/off cycles of the pump
 house.OpTime              : operation time of the pump in s

Outputs

y - Matrix where for timeclass 1 the first row has the annual values.
 For timeclass 2 the 12 rows contain the monthly values.
 Columns:
  1  time of simulation or timeclass start
  2  time of simulation or timeclass end
  3  energy of the heating/cooling system to the house for cooling in Joule
  4  energy of the heating/cooling system to the house for cooling in Joule
  5  transmission losses in Joule (negative value for losses)
  6  ventilation losses in Joule (negative value for losses)
  7  internal gains in Joule (positive value for gains)
  8  solar gain in Joule (positive value for gains)
  9  loss coefficient of the house in W/K
 10  number of on/off cycles of the heating circuit massflow
 11  operation time of the heating circuit massflow in s
 12  average massflow in the heating system in kg/s
 13  mean room temperature in °C
 14  setpoint room temperature in °C
 15  mean ambient temperature in °C
 16  average inlet (flow line) temperature to heating system in degree Celsius *)
 17  average outlet (return line) temperature of heating system in degree Celsius *)
*) The values are only calculated when 'filedetail' is given.

Description

Evaluation of data written by Logger_House. The result of the
function is the energy in annual, monthly, ... values, depending
on the timeclass. For timeclass 1 the first row has the annual values.
For timeclass 2 the 12 rows contain the monthly values.
The outputs are organized in groups: time, energy, efficiency,
operation, average massflows, average temperatures, economics

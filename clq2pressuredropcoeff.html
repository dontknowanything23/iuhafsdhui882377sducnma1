clq2pressuredropcoeff

Path: CARNOT/Basic/Pressure_Drop


Purpose:
The blocks convert the Carnot coefficient q to the pressure drop coefficient ceta.

Description:
Determine the pressure drop coefficient ceta from Carnot q. The pressure drop dp for a given coefficient ceta is:
    dp = ceta * rho/2 * w2    (equation 1)

with:
rho : density in kg/m3
w : velocity in m/s

The Carnot equation for pressure drop is :
    dp = c + l*mdot + q*mdot2    (equation 2)

As the (equation 1) with ceta is quadratic in the velocity (~mass flow), the constant and linear coefficients of the Carnot (equation 2) are zero.

With
    mdot = Vdot*rho = w*A*rho
    w = mdot/A/rho
    dp = ceta * rho/2 * (mdot/A/rho)2
    dp = ceta/(2*A^2*rho) * mdot2 The Carnot pressure drop coeff q is
    q = ceta/(2*A2*rho)

with
    A = pi/4*d2
    ceta = q*pi2*d4*rho/8

For more information see the density block and the velocity block.

Input:

THB : Thermo-Hydraulic Bus
q : Carnot pressure drop coefficient q in Pa/(kg/s)2

Output:

dp_coeff : dimensionless pressure drop coefficient ceta

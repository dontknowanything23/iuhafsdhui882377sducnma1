Stop_Delay

Path: CARNOT/Control

Purpose:
Keep a control signal at a minimum value for a certain time when it changes from a positive value to zero.

Description:
The block keeps an incomming signal at a minimum value when the signal changes from zero a positive value. The switching off is delayed by the specified time.

Input:

In1 : control signal (no unit)

Output:

Out1 : control signal (no unit)

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No

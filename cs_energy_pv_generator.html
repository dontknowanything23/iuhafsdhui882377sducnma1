Function Call

erg = cs_energy_pv_generator([timeclass], [file], [filedetail])

Inputs

timeclass - optional input parameter for the evaluation period
 1 : year (default)
 2 : month
 3 : week
 4 : day
 5 : hour
file - optional character string with the name of the .mat file
 created by the data logger. If not given during function call, the
 function itsself will open an input window for the choice of the file.
 Files written by the Logger_Photovoltaik_Generator have the format
 'pvgeneratoroutput*.mat'
filedetail - optional character string with the name of the .mat file
 created by the enabled data logger in the Logger_Photovoltaik_Generator block.
 This file contain the same information as file but they are stored only
 when there is a DC power generated.
 Files written by the Logger_Photovoltaik_Generator have the format
 'pvgeneratordetailoutput*.mat'
file and, if given filedetail, must contain the following information in timeseries:
 1   pv.SolarEnergyGlobal    irradiated global solar energy in J/m²
 2   pv.Welec_PV             energy produced by the pv generator in J (DC side)
 3   pv.Tcell                cell temperature in degree Celsius
 4   pv.Cycles               number of on/off cycles
 5   pv.OpTime               operation time in s

Outputs

Matrix where for timeclass 1 the first row has the annual values.
For timeclass 2 the 12 rows contain the monthly values.
Columns:
 1   time of simulation or timeclass start
 2   time of simulation or timeclass end
 3   energy produced by the PV generator in J
 4   solar energy irradiated on the PV modules in J/m²
 5   number of cycles
 6   operation time in s
 7   average cell temperature during operation in degree Celsius *)
*) if 'filedetail' is given, the values are evaluated at
   the operation time of the module only

Description

Evaluation of data written by Logger_Photovoltaik_Generator. The result
of the function is the energy in annual, monthly, ... values, depending
on the timeclass. For timeclass 1 the first row has the annual values.
For timeclass 2 the 12 rows contain the monthly values.
The outputs are organized in groups: time, energy, efficiency,
operation, average massflows, average temperatures, economics

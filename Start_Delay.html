Start_Delay

Path: CARNOT/Control

Purpose:
Delay a control signal when it changes from zero to a positive value.

Description:
The block keeps an incomming signal at zero  for a the start delay when the signal changes from zero a positive value.

Input:

In1 : control signal (no unit)

Output:

Out1 : control signal (no unit)

Parameters and Dialog Box:

Characteristics:

Direct Feedthrough : Yes
Sample Time : Inherited from driving block
Vectorized : No
